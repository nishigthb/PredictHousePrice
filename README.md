# House Price Prediction Website
Simple website which aims to predict price of houses located in Banglore

[Dataset](https://www.kaggle.com/amitabhajoy/bengaluru-house-price-data)

## Installation 
Flask

```bash 
  pip install flask
```
  
## Tech Stack

**FrontEnd:** HTML,CSS,Javascript

**Backend:** Flask
